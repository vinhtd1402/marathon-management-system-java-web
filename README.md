# Marathon Management System

## Description
A Java Web application for managing marathon events, runners, and staff operations.

## Features
- Authentication (Login/Logout)
- Runner Management
- Event Management
- Staff Dashboard

## Technologies
- Java Servlet & JSP
- JSTL
- SQL Server
- Apache Tomcat

## How to run
1. Clone project
2. Import into NetBeans
3. Setup SQL Server
4. Run `DataBase_MarathonEvent.sql`
5. Run on Tomcat
